# Sistema Cafeteria Python

Projeto em Python para gerenciar pedidos de uma cafeteria.
O sistema utiliza estruturas de repetição, condicionais e acumuladores para controle de itens pedidos, e exibição de informações ao usuário.

- Estruturas de repetição ('While', 'for')
- Condicionais ('if', 'else')
- Acumuladores e contadores
- Controle de maior valor
- Entrada de dados pelo usuário

## Funcionalidades
- Registro de múltiplos pedidos
- Cálculo do valor total por pedido
- Controle do maior pedido do dia
- Resumo final com estatísticas

## Tecnologias 
- Python 3
